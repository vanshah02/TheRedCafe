The Red Cafe is a fully responsive food delivery website built using HTML5, CSS3, and jQuery, designed to promote healthy, organic dining options across major cities. It features a modern, vibrant layout that guides users from discovering the service's benefits to selecting a subscription plan.

Features:
Hero Section – Engaging landing page with a clear value proposition and call-to-action buttons
Service Highlights – Informative section using icons to showcase benefits like 20-minute delivery and organic ingredients
Visual Meal Showcase – An appetizing image gallery grid displaying a variety of healthy dishes
How it Works – A clear, 3-step visual guide explaining the sign-up and ordering process
City Coverage & Stats – Displays active cities like Mumbai and Bangalore with customer and chef statistics
Subscription Pricing Tables – Comparative pricing cards for Premium, Pro, and Starter plans with feature lists
Interactive Contact Form – A functional form for user inquiries, newsletter sign-ups, and feedback

Tech Stack:
HTML5 – Semantic structure for better accessibility and SEO
CSS3 – Custom grid layouts, responsive media queries, and button styling
jQuery – Used for smooth scrolling navigation and interactive effects
Ionicons – Scalable vector icons for visual enhancements
